# PartialPlus

Go to affezwei.github.io/PartialPlus

Input frequency or note name, calculate overtone/partial, view resulting frequency, note name, and cent deviation. Can adjust A tuning pitch.

One thing to remember, the 1st harmonic/partial/overtone (likewise the -1st subharmonic....) is the same as the fundamental. It is n * 1 or n / 1 which is always n.

Transpose will only change the "Nearest Note" display. This is for players who are following a fundamental from the score in C, but need to check the note in their part. The output will always be in C, as in the score.

Expansion of simple version affezwei.github.io/Overtones
